02. Notefy


A stylish unobtrusive notification system based on libnotify [Linux] or Growl [Mac OSX] depending on how you're living. The original code for Notefy is a fork from jGrowl.

Some major changes from the standard jGrowl code is the addition for icon indicators, and a simplified method for activation.
the notefy() function has four parameters:
"title" , "message" , [optional icon class], [boolean sticky]

04. FaceBox


Facebox is a jQuery-based, Facebook-style lightbox which can display images, divs, or entire remote pages.
It's simple to use and easy on the eyes. For its inclusion in HUI a number of images were changed to css elements. While also altering its size to support a liquid layout.
The window supports static content, remote images, even ajax in external html.
Facebox can be called in functions individually, or more simply, you can include a one liner in your doc ready. Simply put, It's easier to give your Faceboxy links a rel="facebox" and hit them all onLoad.

05. Tooltips

Regular Default Right

Based on the latest 1.3 version of tipTip. The Plugin uses the title attribute just like the native browser tooltip does. However, the title will be copied and then removed from the element when using TipTip so that the browser tooltip will not show up. The Plugin also detects the edges of the browser window and will adjust to stay within the current window size. It supports positioning above, below, to the left and to the right of the element.

08. Flexible Navigation

HorizontalNav

Based on sebnitu.github.com/HorizontalNav and mattkersley/Responsive-Menu.
A horizontal navigation to fit the full width of it's container. There are no requirements to how you style the navigation and adding padding, margins or borders to any of the elements won’t break the effect.
Added benefit that it Turns any <ol> or <ul> into a <select> for mobiles and low browser sizes.

This Page Uses HorizontalNav for its menu. Resize to see in action.

html

<ul>
 <li><a href="#">Navigation Item</a></li>
 <li><a href="#">Work</a></li>
 <li><a href="#">Blog</a></li>

 <li><a href="#">Contact</a></li>
</ul>
