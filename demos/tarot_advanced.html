Tarot Cards Advanced Demo

this demo will show some of the more advanced functionality not included by default. this will allow you to customize the widget to the particular application. This will include changing the content on the fly, and supporting smaller screen sizes.

tarot_basic is the demo from the main index page.

By Leveraging another hui core plugin, antiscroll.js you can easily ensure your deck of cards will work on smaller screens. Touch friendly scroll bars will automatically hide and show to signal overflow and easily move cards in and out of the users visibility.

Resize this Page to Under 800px for automatic scroll effect.

1. Wrap scrollable content with the class ```antiscroll-inner```
2. Wrap the above with the class ```antiscroll-wrap```
3. Include the following Javascript

   $(function () {
     $('.antiscroll-wrap').antiscroll();
   });

In the case of the Tarot Card Widget, the Scrollable Content would be the .tarotDeck class.

If you use the 12 column grid, then the wrap would be the .row you put the widget in.

More Information

You can also you the same layout to create large grids of tiles that stretch across the screen horizontally. Or go all checkboard and have a huge page of cards all laid out.
