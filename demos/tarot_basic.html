Tarot Cards Basic Demo

This is the demo from the main index page. it is included here so viewing the source code is easier.

tarot_advanced has more information.

Forked from github.com/olton/Metro-UI-CSS's Live Tiles. These Cards can can communicate new, real-time information to your user. Cards can show a combination of text and images, and a badge to show status.

Tarot Basics

The default tarotCard size is 150x150 pixels. tarotCard size may be increased with add the next subclasses for tarotCard object: .double(-vertical), .triple(-vertical), .quadro(-vertical). Cards can optionally be Selected, or change background color for tarotCard with classes .bg-color-*.


	  <div class="tarotDeck"></div>
	   <div class="tarotCard"></div>
	   <div class="tarotCard double"></div>
	   <div class="tarotCard selected"></div>
	   <div class="tarotCard bg-color-orange"></div>
	  </div>

Tarot Content

Content can be placed in sub container with class .Tarot-content. Class .Tarot-content has padding 10px from top, bottom and 15px from left, right.

tarotCard Slider

To activate, simply add the attribute data-role="Tarot" to any tarotCard object. They will Animate AUTOMATICALLY!
To set specific parameters such as direction, duration, easing and period you must add param data-param-direction(duration, period) to the tarotCard.

tarotCard status

tarotCard Suit's can be placed in sub container with class .suit or .tarotCard-status. tarotCard status background color can be changed, example with built-in classes .bg-color-* . Or Any Selected Card can set to automatically color shift by adding data-param-shift="true" to the .Selected card.

	<div class="tarotCard">
	<div class="suit">
	...status content here...
	</div>
	</div>

Tile status > badge

A badge can display either a number from 1-99 or a status glyph. badge is positioned in tile status container on the bottom right corner. Metro UI CSS support main Windows 8 badges: activity, alert, available, away, busy, newMessage, paused, playing, unavailable, error and attention as built-in. Default badge background color is blue but may be changed, example with built-in classes .bg-color-*. 

	<div class="tarotCard">
	<div class="suit">
	<div class="badge activity"></div>
	</div>
	</div>
