hui button basics

Button Framework

the button framework contains the core user interaction elements. allowing you to offer everything from a confirmation or yes no button, to a drop down toolbar with @font-face icons.

Sprite Button Demo

All Available Icon Classes in the included PNG Sprite. Simple to use, and great for older systems or quick mockups. Times when editing an entire font file would be inconvenient.


Mouse Over any Icon to see its Class displayed via the tipTip() plugin built into hui.
