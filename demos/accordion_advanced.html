Advanced Accordion Demo

The Flapper Girl menu is fairly simple to setup. The hardest part is remembering the TWO containers around the UL in order to get the stage and everything setup properly.

Animations are handled with CSS3. This means smoother movements across almost all platforms.

If you want a subcategory to start CLOSED add the class .docked to any .title_bar and those LI's will be hidden on startup.

Keep it Tight

The Plugin toggles visibility by grabbing the ID from the .title_bar and using it to match LI's with the corresponding Class. 

...

<li id="bsbs" class="title_bar">Master 1</li>
<li class="bsbs"><a href="#"> . sub 1 . </a></li>
<li class="bsbs"><a href="#"> . sub 2 . </a></li>

...

Targeting the List with some jQuery and CSS3 Hardware Animations will really jazz things up.

The Second Menu has the .push_notification classes applied to spans for numeric push notification style alerts. It also uses the .huicon class to add an icon from the same sprite and class structure used in the Button Icons.
The first menu uses the @font-face icons .fa-icon.icon-magic and can scale to any size without distorting. perfect for retina displays.

var ladyParts = {
	"title":".title_bar",
	"loader":".pageloader",
	"stage":".speakEasy"
};

jQuery("<div> </div>").flapperGirl(ladyParts, callbackGirl);
