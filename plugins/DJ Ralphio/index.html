Single Player

The Easiest Way to Use this Plugin. It Creates a Single Video player from some drop dead simple markup. An Example is featured below.

<div class='singlePlayer'>
  INSERT_VIDEO_ID_HERE
</div>

<script>
  $("WHATEVER").ytSingle();
</script>

Playlist Widget

This plugin is a simple way to include playlists of youtube videos. More information can be found at huement.com/hui and the plugins documentation here: YouTube widget docs. Basically, you construct an unordered list of video ids. A simple html markup example is illustrated below.

<ul class='multiList'>
  <li>Fzj6AFqIoio</li>
  <li>o3Q8RYMSkJo</li>
  <li>7B2UBEn2z7g</li>
</ul>

<script>
  $(".CLASS").DJRalphio("TITLE");
</script>
