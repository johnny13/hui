Tegan & Sara Lists

Custom HUI List Groups with support for Thumbnail Icons and more fancy stuff.

Building Lists

Adding .Tegan to a .list-group container will allow you to use the List group with your standard bootstrap 3 lists. To add an image, attach the .Sara class to any image.

Anytime you add the standard .active to any .list-group-item it will be highlighted.

You can easily mockup blocks by using the Bootstrap coloring scheme of .Sara.primary .Sara.info ... to any .Sara span and it will be highlighted.
