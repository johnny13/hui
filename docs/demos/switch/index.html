Bootstrap 3 Switch

Easily turn checkboxes and/or radio buttons into toggle switches.

Switch It Up

Take way more control over your UI Inputs with this simple plugin.

Add a global declaration for something like $(".switchbox").bootstrapSwitch(); and enable autostart on all switches on all pages in your app.

Bootstrap 3 Switch Demo

  //simple javascript function
  $("[name='my-checkbox']").bootstrapSwitch();
