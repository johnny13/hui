Let DJ Ralphio spin your beats.

Build up simple spinners with complex undertones. Preset limits allow you to trigger messages, disable the input entirely and almost much more. In the example below, you would change out Track1 and quant[1] for unique values.

Spinning Digits.

Customize the #Track1-alert error message to provide quick feedback to your users when limits have been reached.

A disabled input, with optional error message: #Track1-error quickly removes the number spinner from display.

Don't forget you can always sell your grandmas jewelry and go clubbing.

DJ-RALPHIO SPINNER

No Reminder will be sent. This is not recommended.

This has a minimum value of 3. Anything below that triggers the disabled state

  //simple javascript function
  DJ_RALPHIO_SPINS("Track1");

          <!-- Track1 quant[1] -->
          <div class="input-group">
             <span class="input-group-btn">
                 <button type="button" class="btn btn-danger" id="Track1-minus"  data-type="minus" data-field="quant[1]">
                     <span class="fa fa-minus"></span>
                 </button>
             </span>

             <input type="text" id="Track1-error" class="form-control" value="No Reminder" style="display:none" disabled/>
             <input type="text" name="quant[1]" id="Track1-text" class="form-control " value="3" data-limit="3" max="72">

             <span class="input-group-btn">
                 <button type="button" class="btn btn-success" id="Track1-plus" data-type="plus" data-field="quant[1]">
                     <span class="fa fa-plus"></span>
                 </button>
             </span>
          </div>
          <small>
           <p style="margin-top:10px;display:none;" id="Track1-alert" class="alert alert-info alert-dismissable">
             No Reminder will be sent. This is not recommended.
           </p>
          </small>
          <div class="push" style="height:10px;"></div>
          <p class="lead">This has a minimum value of 3. Anything below that triggers the disabled state</p>

DJ-RALPHIO COMPLEX SPINNER

No Reminder will be sent. This is not recommended.

"press and hold" for "boosted" stepping. The value goes from -10 to 1000 in increments of 0.5.

  //javascript plugin
  $("input[name='Track2']").DJRalphio({
    min: -10,
    max: 1000,
    decimals: 2,
    boostat: 5,
    step: 0.5,
    stepinterval: 1,
    maxboostedstep: 50,
    prefix: '<i class='fa fa-rocket'></i>'
  });

  <div class="input-group">
      <input type="text" name="Track2" id="Track2" class="form-control input-number" value="3">
  </div>
