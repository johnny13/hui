hui themes overview

if you are not familiar with .sass or .scss files, you should read up on that real quick. its super easy to get the hang of, and will allow you to build dynamic style sheets really fast.
http://sass-lang.com/

Each of hui's themes are based on a .scss file. This allows you to easily customize the colors of any theme, with out needing to edit much of the css at all. A Node.js module for Sass is included in the hui repository. Otherwise you could install it via ruby gems.

To customize a theme, open any .scss file in your text editor and customize the values. then run your Sass program. If you have Node.js already installed, you can use the GruntJS Gruntfile thats already included by running grunt --force from the hui directory on your system.

Alternatively if you have installed Sass on your system via Ruby Gems, you can run this command: sass /hui_theme_path/dev-theme.scss /hui_theme_path/dev-theme.css
